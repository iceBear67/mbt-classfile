# mbt-classfile

ClassFile parsing/writing support in Moonbit.

Based on [JVMS Ch.4](https://docs.oracle.com/javase/specs/jvms/se24/html/jvms-4.html#jvms-4.5), version Java SE 24.


# Goals
 - [x] Parse ClassFile
 - [ ] Descriptor parser
 - [ ] Signature parser
 - [ ] Write ClassFile
 - [ ] Bytecode \[Dis\]Assembler
